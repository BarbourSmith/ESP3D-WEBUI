Remove Maslow from Frame

Please remove your maslow from the frame. If you need to release tension or set the Z height, use the buttons below.

Before you continue to the next step, if this is your first time on this frame or you are performing another calibration on it, remove any bit from the router, then make sure to lower your Z axis all the way to the bottom until both sides are clicking using the buttons below.

Retract All Belts

When ready, please click Retract all to retract all belts. If a belt does not retract all the way, Click "add tension", then retract all again until they do.

Extend All Belts

When ready, please click Extend all to extend all the belts. They will need to be pulled on to get them to start extending. Continue pulling each belt until it no longer allows you to pull. At the end of all 4 belts, the fan should stop. Please see the user manual if you are not able to do this.

Attach Maslow To The Frame

Please attach all 4 belts of your maslow to the frame. There should be slack enough to do this and the maslow should be roughly in the center of the frame with some slack on the bottom belts. If this is not the case, please enter values in the configuration using the config button and start over.

Apply Tension to Belts

When ready, click Apply Tension to make the belts tight.

Calibrate

If this is a new frame you will need to calibrate. If you have not yet got good fitness you will need to calibrate. If you are happy with your calibration then you can skip this step

If you need to recalibrate you can select Calibrate.
